计算机视觉考前复习

第一章

视感觉：从分子的层次和观点来理解人类对光的反映的基本性质。

视知觉：论述人们从客观世界接收到视觉刺激后如何反应以及反映所采用的方式。

研究方法：仿生学、工程方法。

目标：

  • 建立计算机视觉系统完成各种视觉任务。
  • 以对计算机视觉的研究来探索人脑视觉工作机理。

图像：辐射强度模式的空间分布。

类别：

  • 模拟图像：从连续的客观场景直接观察到、一个用二维数组来表示的图像
  • 数字图像：把连续的模拟图像在坐标空间XY和性质空间F都离散化了的图像。

像素之间的关系：

  • 邻接：对于两个像素p、q，如果p在q的邻域中，则说两个像素邻接。
  • 连接：p和q邻接，且灰度值满足某个相似准则。
  • 连通：互相不直接连接，但是都在某个像素的邻域内，并且都满足某个像素准则。
  • 通路：两个像素之间有一系列连接的像素使得p和q连通，则这些像素构成p和q之间的通路。

像素间的距离：

欧氏距离、街区距离、棋盘距离。（计算题）

第二章

采集装置：传感器、数字化器件

CCD摄像机：具有非常快的快门速度

CMOS摄像机：低功耗、小尺寸、总体成本低。

CID摄像机：随机访问，不会产生图像浮散。

基本性能指标：

线性响应、灵敏度、信噪比（越高越好）、阴影（不均匀度）、快门速度、读取速率

重合模型：摄像机坐标系统和世界坐标系统重合。

通过投影变换将世界坐标系的东西投影到摄像机坐标系上。

4-点映射。

归一化摄像机：**焦距为1的特定摄像机**，也指一种简化的重合模型。

当焦距减半，成像尺寸也会缩减到一半

当传感器密度减半，成像尺寸也减半。

分离模型

图像平面（原点）与世界坐标系统的位置偏差用矢量D表示，分量有Dx、Dy、Dz

摄像机的扫视角：x和X轴的夹角

摄像机的倾斜角：z和Z轴的夹角。

若将XY平面视作地球的赤道面，Z轴指向北极，那么扫视角对应经度，倾斜角对应纬度、

亮度成像模型

图像采集的过程可以看做一个将客观景物的光辐射强度转化为图像灰度的过程。

照度：
$$
E = k \frac{cos i}{d^2}= \frac{kh}{d^3}
$$

空间分辨率：即数字化的空间采样点数（与采样点数有关，采样越密，分的越清）

幅度分辨率：即采样点值的量化级数（量化灰度级数，级数越多，分辨率越高）

结构光法：

  • 直接获得深度信息。

  • 利用照明中的几何信息来提取景物自身的几何信息。

  • 成像高度包含了3D的深度信息。

  • 也包含了物体的沿y方向的厚度信息。

什么是摄像机的标定？

根据一组基准点计算出相机的内外部参数的过程。

  • 外部参数：即摄像机姿态参数。**自转角、进动角、章动角**、Tx Ty Tz
  • 内部参数：即摄像机自身参数。焦距、镜头失真系数k、不确定性图像尺度因子u、图像平面原点的计算机图像坐标。
  • k：镜头径向畸变引起的误差系数
  • u：图像获取硬件和摄像机扫描硬件之间的误差。

如果u已知，只需要一组共面基准点；如果未知，需要一组不共面的基准点。

目的是什么？

让摄像机的坐标系统和世界坐标系统对齐。

客观场景->数字图像的四步：☆☆☆

  1. 世界坐标到摄像机3D坐标的变换。（分离模型变为重合模型）
  2. 从摄像机坐标到无失真成像平面理想坐标（投影变换矩阵）
  3. 从无失真成像平面理想坐标到受镜头畸变而偏移的实际成像平面坐标。
  4. 从成像平面坐标到图像坐标的变换。（连续坐标变为离散坐标）

第三章

几何失真校正：空间变换、灰度插值。

图像增强：直方图修正——均衡化、规定化。P51

消除噪声的方法：参与运算的像素越多的方法越好。

第四章

一阶微分算子给出梯度，也称作梯度算子（计算题）

分别计算x和y方向的两个偏导分量

梯度就是两个方向上的偏导。

prewitt sobel robert

二阶导数算子：拉普拉斯算子、对噪声敏感、模板中正邻负

马尔算子：在拉普拉斯算子基础上，先平滑，再检测。过零点作边缘点。

坎尼算子：满足三个准则↓

  • 信噪比准则
  • 定位精度准则
  • 单边缘响应准则

susan算子：

核同值区（USAN）原理。

  • 直边缘：usan一半。
  • 交点处：usan约1/4

利用usan面积的变化可检测边缘或者角点。

特点：

  • 不需要积分运算
  • 响应效果随着边缘的平滑或模糊而增强
  • 提供不依赖于模板尺寸的边缘精度（控制参数简单）

哈夫变换;

点线对偶性：图像空间共线的点对应参数空间相交的线。

竖直的线使用极坐标方程，点-正弦曲线对偶性。

第五章

轮廓搜索：图搜索、动态规划。

动态规划：使用启发性知识，**步骤见P96**

并不能保证发现区局最小代价通路。

阈值分割：

大于阈值取1，小于阈值取0，最终得到二值图像。

关键在于选取合适的阈值：

全局阈值：仅根据图像本身f(x,y)来选取。

局部阈值：根据f(x,y)和像素周围局部区域性质q(x,y)

动态阈值：根据f(x,y)和q(x,y)，以及像素坐标位置(x,y)选取、

前两种阈值可称为固定阈值。

全局阈值的选取：

  • 极小值点阈值——找到曲线中的谷。
  • 最大凸残差——凸包与实际差值最大的位置。

特征空间聚类：

结合了阈值化分割和标记。

k-means聚类：将一个特征空间分成k个聚类。

  1. 任意选k个初始类均值。
  2. 在每一次迭代，将每个特征点付给——类的均值与其最接近的类。
  3. 所有点都分类之后，更新所有类的均值。
  4. 如果类的均值不再随着迭代次数变化，则说明算法收敛。

第六章

链码：4-方向链码、8-方向链码。

不同起点有着不同的链码。

归一化链码：先得到一个任意起点的链码，再将这个序列循环，直至得到其值最小的序列。

平移不会改变链码，但是旋转会改变。

链码旋转归一化：链码的差分码。（具有旋转不变性）

计算差分码：x - y，总要确保 y 大于 x ，如果 y 小于 x， 就将其加上链码的方向数再算。

缝隙码：连接两个相邻轮廓像素的中心。

凸包：由边界凸起形成线段并将原形状完全包括的最小多边形。

凸残差：凸包减去原形状。

分段点：凸包边界和原形状一般都有重叠部分，重叠部分和凸包边界的分段点就是。

边界标记：

  1. 距离为角度的函数
  2. fai-s曲线
  3. 斜率密度函数
  4. 距离为弧长的函数（弧长与角度有关）

四叉树：

目标节点（白色）、背景节点（黑色）、混合节点（黑色）。

每个混合节点都还可以再分四个节点。

一个n级的四叉树，节点总数N最多有
$$
\frac{3}{4}4^n
$$

外接盒：包含目标区域的最小长方形。

围盒：最小包围长方形。（MER）

凸包：包含目标区域的最小凸形。

骨架：

如果对于P，在边界B上有两个及以上的点到P的距离最小，则认为P是骨架点。（与边界点距离最小）

骨架与距离有关，也就与距离度量有关。

计算骨架：

  1. 标记并除去满足以下条件的点：
    1. 2 <= N(p) <= 6
    2. S(p) = 1
    3. p2 * p4 * p6 = 0
    4. p4 * p6 * p8 = 0
  2. 标记并除去满足以下条件的点：
    1. 2 <= N(p) <= 6
    2. S(p) = 1
    3. p2 * p4 * p8 = 0
    4. p2 * p6 * p8 = 0
  • 条件1为了保护线段端点或者深入区域的点
  • 条件2保护了宽度为单个像素的线段（边界）
  • 条件3、4保护了右、下角，左、上角。

边界长度：包围区域的轮廓的周长

4-方向连通边界：要求点的8邻域内有不属于R的点。

8-方向连通边界：要求点的4邻域内有不属于R的点。

边界长度：

边界上相隔最远的两个点之间的距离。（主轴、长轴）

短轴：与长轴垂直且是边界间最远两点的线段。

边界形状数：基于链码，是值最小的（链码的）差分码。

也可以是链码的差分码的归一化值。

阶数：形状数的长度。随着阶数的增加，对应的可能的形状数也会很快增加。低阶形状数对应的形状有限。

计算:（计算题）

  1. 求出围盒
  2. 划分围盒为多个小正方形，将与边界有50%以上面积重合的小正方形选为边界的模拟。
  3. 根据小正方形组成的边界的模拟来得出链码，然后得到差分码，然后得到形状数。

轮廓形状矩阵:P127

可看做是一个刻画目标轮廓的矢量，记录了轮廓上各点的相对位置。

  1. 获取轮廓

  2. 对轮廓采样得到离散点

  3. 划分扇区，划分圆环。

  4. 对每个点，将其放在圆的中心，记录其他点在扇区、圆环中的分布情况。

    （矩阵行：环数，矩阵列：扇形区数）

  5. 对每个点的统计得到的就是形状矩阵。

区域不变矩：（计算题）

区域拒是用所有属于区域内的点计算出来的。

p+q阶矩 = 所有求和【x^p * y^q * f(x,y)】

区域矩唯一的被f(x,y)所确定。

中心距：

拓扑描述符：（计算题）

与尺寸无关，不收畸变变形的影响。

欧拉数：E = C - H。 C：联通组员个数。 H：孔洞数。

欧拉数描述的是区域的连通性。

欧拉公式：V - B + F = E = C - H。 欧拉数可以取负数。
