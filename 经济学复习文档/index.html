需求

需求——表示在某一定时间内，在各种可能的价格水平下，个人愿意并且有能力购买的某种商品的数量。

（多种价格对应多种需求量的情况）

需求量——在商品某一价格下，对其需要的数量。

**影响需求的因素**：

  • 产品自身价格
  • 偏好
  • 收入
  • 相关产品的价格
  • 预期
  • 人口
  • 其他：法律政策等。

**吉芬商品:**商品价格下降，消费者购买少；而价格上涨时，需求量反而上升。（买来投资）

替代品：可以替代另外的商品的商品

互补品：消费者为了实现某种功效而需要一起消费的商品（订书机、订书钉）

正常品：随着消费者收入增加需求增加的商品（好吃的）

劣等品：随着消费者收入增加需求减少的商品。

需求曲线：表示一种物品的价格和需求量之间关系的曲线。（一般是负相关）【横轴需求，纵轴价格】

需求定理：其他条件不变，商品的价格上升，对其的需求减少；价格下降，对其的需求增加。

需求曲线的左移右移：

  • 左移——同价格的需求减少了
  • 右移——同价格的需求增加了

需求量的变化——商品本身价格引起的需求变化。（需求曲线点的移动，不改变曲线）

需求的变化——除价格之外的因素引起的需求变化（或同一价格水平下需求量的改变）。（曲线左右移）

需求的变化必定引起需求量的变化，但是需求量的变化不一定引起需求的变化。

供给

供给：在一定时期内，在一定价格条件下，企业愿意并能够生产（销售）某种商品的数量

影响供给的因素：

  • 产品自身价格
  • 技术状况
  • 生产要素的价格
  • 其他商品价格
  • 预期
  • 厂商的数目
  • 政府税收

供给曲线：表示物品价格与供给量的关系。【纵轴价格，横轴供给量】

供给规律：认为在其他条件不变时，一种物品价格上升，该物品供给量增加的观点。

供给量的变动：商品价格变动引起商品供给数量的变动。供给量的增加是因为价格的上升。(需求上升引起的**点的移动)**

供给的变化：同一价格下，商品的供给量发生了变化。

  • 左移，供给减少，需求减少。
  • 右移，供给增加，需求增加。

市场均衡与供求规律

市场均衡即供求相等时的市场关系。需求曲线与总计曲线的交点对应着均衡价格、均衡数量。

任何偏离均衡点的一点都是市场失衡。市场有自发的力量推动市场均衡的恢复。

限制性价格政策——限制价格（最高价）。——房价

支持性价格政策——支持价格（最低价）。——粮食价格

谷贱伤农：

  • 需求弹性是指在一定时期内商品需求量的相对变动对于该商品价格的相对变动的反应程度

  • 需求价格弹性 = 需求量变动百分比 / 价格变动百分比

    • 衡量物品需求量对其**价格变动反应程度**
    • 衡量买者需求的价格敏感程度

  • 需求曲线越**平坦**，该商品越富有价格弹性，需求弹性越小

  • 越日常、刚需的东西，需求弹性越小（鸡蛋0.1、大米0.5、住房0.7、威士忌4.4）

  • 生活必需品、衣食住行适合薄利多销。

  • 价格弹性>1，需求弹性小，可以薄利多销。

  • 需求弹性、价格弹性基本**负相关**。

供给弹性

度量某一商品供给量变化率对自身价格变化率反应程度。

供给价格弹性 = 供给量变动百分比 / 价格变动百分比

总结：

  • 需求曲线越平坦，价格弹性越高、需求弹性越小。（生活必需品）

  • 一个商品如果很少有替代品，价格弹性小，需求弹性大。

  • 支持性政策会导致产品过剩，

第三章 消费者

  • 无差异曲线及其特点
  • 预算线（判断、名词解释）

想要什么？

效用：消费者从商品和服务的消费中得到的满足程度称为效用

  • 不是客观用途（不是商品的使用价值）
  • 是个人的心理感受
  • 同一商品的效用的大小因人、因时、因地 而异。

关于效用的两类观点：

基数效用论：效用可以用数值计算，可以加总求和，提出了边际效用递减规律。

序数效用论：效用作为一种心理现象无法计量，只能表示程度的高低、顺序。

**边际效用递减规律**：

是指在一定时间内，在其他商品的消费数量保持不变的条件下，随着消费者对某种商品消费量的增加，消费者从该商品连续增加的每一消费单位中所得到的效用增量即边际效用是递减的。

  • 一直冲，冲得越多爽的越少。
  • 当消费者已拥有了所有的某种想要的商品，那么此时该商品的边际效用为0。

产生原因：生理或心理的原因、商品不同用途的重要性不同

无差异曲线：在满足消费者效用水平不变的情况下，消费者消费两种商品的消费束代表的点的轨迹。

（线上任一点，代表两种商品的不同组合，暗示任一点带个消费者的效用都相同）

  • 代表了消费者的偏好。

  • 消费束——是物品的一个特定组合。

  • 离原点越远，代表效用水平越高。

  • 任意两条无差异曲线不能相交。

  • 在多数情况下，无差异曲线是凸向原点的。

  • 相近替代品的无差异曲线不是很弯曲。

  • 相近互补品的无差异曲线非常弯曲。

  • 无差异曲线上一点的**斜率是消费者的边际替代率**。

边际替代率（MRS）：在维持效用水平不变的前提下，消费者增加某一商品消费数量时所需要放弃的另一种商品的消费数量。（消费者愿意用一种物品换另一种物品的比率。）

  • 计算：MRS = 无差异曲线的斜率。 MRS = △y / △x。

能买什么？

预算约束线：预算线（Budget line）：在既定价格水平下，消费者用给定的收入可能购买的各种商品组合点的轨迹。有时称预算约束（budget constraint）或者消费可能线、价格线。

P1Q1 + P2Q2 = I 【P价格，Q数量，I 总收入】

找到分别全买A、B商品的两个点，相连就是要求的预算约束线。

  • 收入增加使得约束线向右上方移动，减少使得约束线向左下角移动。

最优选择

在消费决策中，作为理性人，你的选择也应该是追求消费的均衡

消费者必然追求消费效用的最大化，但作为现实中的个人，消费者又必须受到消费条件的制约。

恩格尔系数：食品支出占个人消费指出总额的比重。【越低越有钱】

收入效应：

  • 收入变动对消费的影响。随收入增加而消费量增加的商品为正常物品，收入效应为正。
  • 随收入增加而消费量减少的物品是低档物品（劣等品），收入效应为负。

替代效应：当两种商品之间有替代关系时，在收入不变时，其中一种商品的价格发生变动，消费者对原来的消费组合调整。

价格效应 = 收入效应 + 替代效应

第四章生产者

  • 长期与短期的区别（判断）
  • 总产量和边际产量的关系
  • 规模报酬递分？
  • 总成本和边际成本怎么计算？
  • 经济利润、显性成本和隐性成本

企业：厂商或生产者，是指能够做出统一的生产决策的单个经济单位。

必须具备的三个特征：

  • 独立自主经营
  • 自负盈亏
  • 产权明晰

通常有三种组织形式：个人企业、合伙制企业和公司制企业。

  • 个人企业：单个人独自经营的厂商组织
  • 合伙制企业：两个人以上合资经营的厂商组织
  • 公司制企业：按照公司法建立与经营的具有法人资格的厂商组织。

企业追求——利润最大化

利润 = 总收入 - 总成本

生产要素：资源、资本、劳动、企业家的才能。

生产函数：表示在一定时期内，在技术水平不变的情况下，生产中所使用的各种生产要素的数量与所能生产的最大产量之间的关系。

为了简化分析，通常假定生产中只使用劳动和资本这两种生产要素。Q = f ( K , L ) = aL + bK {线性生产函数}

【Q为产出、L为劳动力、K为资本。】

边际产量——在其他投入量不变的情况下，增加一单位投入所引起的产量增加。（MPL = △Q / △L）

  • 随着L增加，MPL逐渐减小。
  • 随着L增加，生产函数走势越来越平坦。

企业收益分析

短期生产——投入的生产要素根据变或不变的属性，进行分类。

  • 可变投入：劳动、电力、燃料等消耗性要素。
  • 不变（固定）投入：土地、厂房、机器设备。

长期生产——**所有的生产要素都是可变的**。

注意：

  • 固定投入和可变投入的区别仅限于短期生产。
  • 厂商只有在长期生产才面临不同生产函数的可能
  • 短期长期的划分并非依据时间
  • 对于不同产品的生产，短期生产和长期生产的时间不同。

短期生产中的生产规律

总产量：生产者投入一定的生产要素之后，得到的产品的总和。

边际产量：再正价某种生产要素带来的总产量的增加量。

短期生产中，边际产量先增加后减少——边际收益递减。

**边际报酬递减规律**：短期生产中，在技术水平不变的条件下，不断增加某种生产要素的投入，当投入的数量达到一定的数量后，再增加该要素带来的边际产量是递减的。

长期生产中的生产规律：

规模收益（规模报酬）——在技术水平与要素价格不变的情况下，所有生产要素都按同一比例增加，引起的产量的变动状态。

  • 规模收益不变——产量增加的比例 = 各要素投入量增加的比例
  • 规模收益递增——产量增加的比例 》 各要素投入量增加的比例
  • 规模收益递减——产量增加的比例《 各要素投入量增加的比例

规模经济：

  • 规模经济——长期平均总成本随着产量的增加而减少。【越产越多，越花越少】
  • 规模不经济——长期平均总成本随着产量的增加而增加。【产的多，花的也多】

随着规模增大，由规模经济进入不规模经济。规模收益先递增后不变，最后递减。

企业成本分析

成本：是企业支付给生产要素的报酬，也即生产一定数量产品所耗费的指出。

显性成本：又称可见成本，有些生产要素在购买时需要支付报酬，一般用货币来支付，即货币成本

隐性成本：有些生产要素报酬不需要立即支付，或合同规定必须支付，却实实在在发挥着生产作用

机会成本（显性成本+隐性成本）：生产要素用于目前这种用途时所放弃的其他用途的最高收入。

会计成本：会计学上的成本——**计算显性成本，忽略了隐形成本**。

会计利润：不考虑隐性成本，总收益 - 总显性成本

经济利润：总收益 - 机会成本

沉没成本：已经付出的永远无法收回的成本（如时间）

  • 沉没成本不应该影响下一次决策。

经济学家衡量成本包括机会成本，但会计只关注显性成本而忽略隐性成本。

可变成本（VC）：短期内随产量变化而变化的那部分生产要素报酬。【工人的工资】

固定成本（FC）：短期内不随产量变化而变化的那部分生产要素报酬。【土地的价格、机器的价钱】

总成本：是可变成本和固定成本之和。**TC = VC + FC**。

平均成本：平均生产一单位产品所需的成本——AC。

边际成本：增加一单位产量时所需增加的成本。——MC。

  • MC = dTC / dQ = **△TC / △Q**。

第五章 市场结构

市场结构：完全竞争、垄断竞争、寡头垄断、垄断

划分尺度：生产者数量的多少（上述递减）、生产者所生产的商品的差异性

完全竞争市场：

不存在丝毫垄断因素，竞争可以充分展开，市场机制在资源配置方面的作用不受到任何阻碍和干扰

特点：

  • 生产者生产的商品是完全相同的
  • 买方和卖方都是价格接受者
  • 企业可以自由地进入和退出市场
  • 市场信息是完全的

利润化最大条件：

MR = MC，边际利润等于边际成本。

利润 = 总收益 - 总成本

总收益 = P * Q 总成本 = ATC * Q

所以利润 = （ P - ATC）* Q

在生产成本高于市场价格，即会产生亏损的情况下，商家还是要生产和售卖，通过一直做生意来逐渐减小亏损。

AVC平均可变成本——人工费用 AFC平均固定成本——其他材料、房租等成本

如果AVC很高的话，生意是做不下去的。

残酷的完全竞争：

利润被无情地压缩是完全竞争行业的一个显著特点

**只有当价格与ATC（平均总成本）相等时，进入和退出才会结束**。

垄断市场：

完全垄断：又称垄断或者独断，是指整个行业中只有唯一的一个厂商的市场类型

产生原因：

  • 垄断资源：当某行业中至关重要的资源或其他投入要素被某厂商所控制，其他厂商无法获取值这资源，也就不发进入这一行业。
  • 政府管制：垄断的产生是因为政府给予一个人或一个企业排他性地出售某种物品或劳务的权利
  • 自然垄断：当一个企业能以低于两个或更多企业的成本为整个市场供给一种物品或劳务时，就是自然垄断。
  • 专利和版权（比如冰墩墩）

一级价格歧视：对每个消费者，相同的商品的售价都不一样

二级价格歧视：按照消费者的消费数量来确定价格。

三级价格歧视：将市场划分为多个不同的子市场，并将同一种商品在不同的子市场上按不同价格销售。

寡头垄断

在某一产业中只存在少数几个卖方的市场组织形式。

基本特征：

厂商数量极少、商品可能同质，也可能有差别、市场存在较高的进入壁垒、厂商之间相互依存。

垄断竞争：许多厂商出售相近但非同质的、而是具有差别的商品的市场

基本特点：

  • 存在较多厂商、彼此之间竞争激烈
  • 各厂商生产有差别的同种产品、产品之间是非常接近的替代品
  • 厂商进入、退出市场都比较容易
  • 厂商行为相互独立

小结：

垄断可以促进新技术的出现、完全竞争可以让资源得到最大化的利用。

如何衡量经济状况

  • 什么是GDP？
  • GDP怎么核算？
  • 名义GDP与实际GDP

GDP：国内生产总值（Gross Domestic Product):在某一既定时期内一个经济社会运用生产要素所生产的全部最终产品（物品和劳务）的市场价值。

包含了以下几方面的内容：

  • GDP是价值概念
  • GDP测度的是最终产品的价值
  • GDP测度的是**现期生产**的产品
  • GDP局限于一国地理范围之内
  • GDP是一定时期内生产的最终产品的市场价值

【家庭劳动、自给自足不算入GDP】

最终产品：同期内不再加工，可供最终使用与消费的产品

中间产品：用于再出售而供生产别种产品用的产品

同一种产品可以是最终产品也可以是中间产品

只有最终产品才算入GDP

商品（最终产品）计入生产当年的GDP

GNP（Gross National Product）：是一个国民概念，一定时期内某国国民拥有的生产要素所生产的最终产品和劳务市场价值。

GDP只是计算国家领土范围之内的，GNP计算的是国籍层面上的。

GDP可以帮助领导人判断国家的经济情况，需要做出怎样的经济控制措施。也能比较国与国之间的经济实力。

GDP如何核算？

  1. 生产法：从生产者角度来计算GDP，生产总价值 - 生产中投入的中间产品的价值。

​ GDP = 各行业增加值之和 = 各行业总产出 - 各行业中间消耗

  1. 支出法：

    • 核算在一定时期内整个社会购买最终产品的总支出，即最终产品的售价来计量GDP。

    • 包括**消费、投资、政府购买、净出口**

      • 居民消费：耐用消费品、非耐用消费、劳务（不包括买房子造房子）

      • 投资：增加或更换资本资产的支出，固定资产投资+居民住宅投资+存货投资

        【债券、股票不计入投资】

      • 政府购买：各级政府购买物品和劳务的支出，包括公务员的薪水和用于公务的支出。

        政府支出 = 政府购买+政府转移支付+公债利息。（后面两个不计入GDP）

      • 净出口：国外购买国内生产的物品与国内购买外国生产的产品之间的差额。

  2. 收入法：（最复杂）

    用要素收入，即企业生产成本来核算GDP

    包括：

    1. 工资、利息、租金等生产要素的报酬

    2. 非公司企业主收入，如小店铺主的收入

    3. 公司税前利润，包括公司所得税、社会保险税、股东红利、公司未分配利润等。

    4. 企业转移支付和企业间接税

    5. 资本折旧。

    GDP = 工资+利息+租金+利润+间接税+企业转移支付+折旧。

名义GDP与实际GDP

名义GDP：一个经济体在一年内用当年价格计算的最终产品的市场价值。

实际GDP：一个经济体在一年内生产的以某年作为基期价格计算的全部最终产品的市场价值。

GDP平减指数：名义GDP和实际GDP的比率。反映通胀的物价指数。（名义 / 实际）

我国名义GDP和实际GDP每年都在增加，但是由于物价水平不断上涨，两者差距越来越大）

CPI： 消费者物价指数

是度量一组代表性消费商品及服务项目的价格水平随时间而变动的相对数，是用来反映居民家庭购买消费商品及服务的价格水平的变动情况。

CPI与GDP的差别

  • GDP衡量所有所有产品和服务，CPI只衡量消费者购买的。
  • GDP只包括国内生产的产品，但是CPI受进口品的影响。
  • CPI给不同产品分配固定的权重，而GDP分配变动的权重。

经济增长

消费

  • 消费通常指最终消费，主要包括了居民消费和政府消费。
  • 直接推动经济增长，创造生产动力，进一步刺激投资需求。

投资

  • 经济学上的投资不同与一般意义上的投资，并不包括对股票证券的投资。

  • 投资可以增加社会总供给，扩大社会生产能力。

出口

出口带动国内产业的发展，提供大量就业岗位，提高居民收入水平，扩大消费需求。

如何看待GDP？

GDP的局限性：

  1. 没有考虑非市场性活动
  2. 没有考虑环境破坏活动
  3. 没有考虑收入分配
  4. 不是衡量福利的完美标准

如何保持长期经济增长？

  1. 物质资本的增加
  2. 人力资本的增加
  3. 技术和研发
  4. 政治稳定和产权保护

小结：

还有局限性和对长期经济增长的保证。

补充：

转移支付——指政府或企业无偿地支付给个人以增加其收入和购买力的费用。

第七章 失业与通货膨胀

  • 失业的定义及判断
  • 失业率的计算
  • 充分就业、自然失业率
  • 失业的类型
  • 失业的影响
  • 通胀的定义
  • 通胀的影响
  • 通胀的成因
  • 菲利普斯曲线

失业

在当前工资水平下愿意工作的人无法找到工作的一种形式。

**失业者的三个条件**：

  • 符合劳动年龄（16-65）
  • 有工作能力
  • 愿意工作但没有工作

总人口——成年人口、未成年人口

成年人口——劳动力人口、非劳动力人口

不列入劳动者的：

  1. 军队人员
  2. 在校学生
  3. 家务劳动者
  4. 退休或者因病退职人员以及丧失劳动能力、服刑犯人等不能工作的人员
  5. 不愿意工作的人
  6. 在家庭农场或家庭企业每周工作少于15个小时的人员

失业的计算

劳动力人口 = 就业人口+失业人口

失业率：失业者占劳动力的百分比

参工率：劳动力占成年人口的半分比

失业的类型

按劳动力市场结构分为：**摩擦性失业、结构性失业、周期性失业**

摩擦性失业：

经济中由于劳动力的正常流动而引起的失业。

原因：既有自愿失业的因素（不满待遇），也有非自愿失业的因素（认为没有合适的岗位），但社会上存在适合它们的工作岗位，只是暂时未进入就业状态。

特点：经济中失业人数与职位空缺数额相当。摩擦性失业是由于劳动者流动不充分以及劳动力市场信息不灵等原因引起的。

摩擦性失业通常起源于劳动力的供给方

结构性失业：
是劳动者的技能与空闲的工作岗位不匹配造成的失业。

原因：由于社会中经济结构的变化进而引起劳动力需求结构的变化，而劳动力供给的调整往往却滞后于劳动力需求结构的变动，因此导致结构性的失业。

特点：失业和职位空缺大致相等，但劳动力的供求长期不一致，往往属于非自愿失业。通常持续时间较长。

摩擦性失业与结构性失业都属于自然失业

某个行业不景气导致的失业是结构性失业

周期性失业：

是指由于**总需求水平不足，导致企业生产减少，开工不足，由此带来的劳动力总需求不足而引起的短期失业，又称需求不足型失业**。

原因：与经济周期相联系。经济繁荣时期总需求水平高，则失业率低；相反经济萧条时期总需求不足，则失业率高。

特点：周期性失业基本上属于非自愿性失业。

周期性失业通常起源于劳动力需求不足。

摩擦性失业和结构性失业在劳动力市场处于均衡时也存在，而周期性失业时劳动力市场必定处于非均衡状态。

康波周期：

  1. 衰退阶段
  2. 复苏阶段
  3. 过热阶段
  4. 滞涨阶段

所有的劳动力都有工作才是充分就业吗？

充分就业：除了摩擦性失业和结构性失业外，所有愿意工作的人都在从事工作的状态。

自然失业率：充分就业时的失业率 = 摩擦性失业率 + 结构性失业率

自然失业率是劳动力市场和商品市场处于均衡状态时的失业率，也是能够长期持续存在的最低失业率。

失业的影响

  • 影响GDP
  • 影响社会（公众心理）安定
    • 短期，小影响多数人
    • 长期，大影响少数人

造成的经济损失：失业会对社会经济造成重大影响而带来巨大损失，其中，最明显地表现为实际GDP的减少。

奥肯定律：失业意味着生产要素的非充分利用，失业率的上升会伴随着实际GDP的减少，失业率每上升1%，实际GDP大约减少潜在GDP的2%。

失业的社会成本：失业不仅会减少当事人的收入，还会给他们造成很大的心理伤害，从社会方面看，事业的影响虽然无法用货币单位表示，但这种影响可能是非常巨大的。

失业对分配的影响：失业成本的承担很不平衡，如果将失业划分为短期失业和长期失业的话，短期失业与长期失业对收入分配所造成的影响完全不同。

  • 短期失业（主要指摩擦性失业）：由于事业的持续时间很短，且人员流动频繁，失业成本将由众多人承担，因而每个人承担的损失就很小。
  • 长期失业，则主要由劳动力市场结构因素或工资刚性形成，失业成本主要由少数人承担，每个人承担的损失较大。另一方面，失业者可能停止纳税，还有可能得到失业保险的好处或其他政府转移，则事业的损失由就业者（纳税人）承担。

通货膨胀

概念：是指在**纸币流通的制度下，在一定时期内，由于货币发行量超过流通中实际货币需求量而引起的物品和劳务价格的普遍持续性**上涨。

使用金银流通的社会一般不会发生通胀，只会通缩，因为金银的量太少了，跟不上经济发展的需求。

衡量：

衡量通货膨胀的指标由物价指数（价格指数）来表示，称为通货膨胀率。

西方主要使用消费者价格指数CPI、生产者价格指数PPI、GDP来衡量。

分类：

按价格上涨速度或是通货膨胀率的高低：

  • 爬行的通货膨胀：1% ~ 3% 有利于经济
  • 温和的通货膨胀：3% ~ 6%
  • 严重的通货膨胀：6% ~ 9%
  • 奔腾/急剧的通货膨胀：10% ~ 100%
  • 恶性/超速的通货膨胀：三位数以上，价格水平失控（津巴布韦）

按对价格影响的差别：

  • 平衡的通货膨胀：商品和劳务的价格同步上升
  • 不平衡的通货膨胀：商品和劳务的价格上升比例不同

按照人们对通货膨胀的预料程度：

  • 预料得到的通货膨胀（惯性通货膨胀）
  • 未预料到的通货膨胀

通货膨胀的表现形式不只有物价上涨，限购也是一种（限制人们消费）

经济效应

产量效应：短期内，未预料的、温和的通货膨胀会促进经济增长。长期来看，通货膨胀与产量之间没有必然的联系

收入再分配效应：其使一部分人实际收入增长，另一部分人实际收入减少。（扣除了通胀影响之后）

通胀利于债务人（欠钱的人）、雇主、政府；不利于债权人（债主）、工人、公众。

名义上的利率、收入，扣除掉通胀率的影响就是实际的利率、收入。

财富再分配效应：非预期的通胀还会引起财富的再分配效应。对拥有货币财产与非货币财产者的财产净值的影响。

在通胀期，非货币资产能保值，而货币资产不能保值。

通胀的成因：

  1. 需求拉上的通胀：需求增长引起的通胀
  2. 成本推动的通胀：成本的提高引起的通胀。
  3. 需求拉上余成本推动混合的通胀
  4. 结构性通胀：产业和部门之间的“看齐”行为，导致一般物价水平上涨引起的通胀（深圳）

只要需求增加，就会发生通胀吗 ？ 不是

是不是只有充分就业时才会出现通胀？ 不是

通胀的防范和治理

  1. 防止一次性通胀演变为持续性通胀
  2. 抑制持续性通胀
  3. 治理通胀的具体政策（对策）
    1. 需求政策：采取紧缩需求的政策就能取得一定的效果。
    2. 收入政策：限制提高工资和获取利润，以抑制成本推动型通胀
    3. 收入指数化政策：各种收入部分或全部地与物价指数相联系。
    4. 供给政策：增加生产和供给。
      1. 减税
      2. 削弱政府开支增加幅度
      3. 限制货币增长率
      4. 改善劳动市场结构的人力资本政策

菲利普斯曲线

最初的曲线:当失业率较低时，货币工资增长率较高；当失业率较高时，货币工资增长率较低，甚至负数。

改进后：**刻画了失业率和通货膨胀率之间的负相关系**。

菲利普斯曲线的发展三阶段：

  • 通胀和失业率之间存在一种稳定的负相关系
    • 提高通货膨胀，可以降低失业率
    • 降低通货膨胀，失业率会升高
  • 解释短期菲利普斯曲线与长期菲利普斯曲线之间的根本区别
    • 短期菲利普斯，附加了预期
    • 适应性预期：人们根据过去的经验来形成并调整对未来的预期。
    • 缺陷：忽略了工人对通货膨胀的预期。
    • 长期菲利普斯，是一条垂直于自然失业率水平的垂线。
  • 失业率和通胀之间不存在规律的替代关系。

菲利普斯曲线的提出

当失业率较低是，货币工资增长率较高，反之，当失业率较高是，货币工资增长率较低，甚至为负数。

改良

失业率与通胀之间的负相关系，表明失业率与通胀之间存在着替换的关系

通胀率 = 工资增长率 - 劳动生产率增长率

附加预期的菲利普斯曲线——短期菲利普斯曲线（向右下倾斜）

适应性预期：人们会根据过去的经验来形成并调整对未来的预期。

曲线的缺陷：忽略了工人对通胀的预期，得到了短期菲利普斯曲线

曲线的政策含义：在短期中引起通胀率上升的扩张性财政政策与货币政策是可以起到减少失业的作用的。

调整总需求的宏观经济政策在短期是有效的。

长期菲利普斯曲线——表明通货膨胀和失业率之间不存在相互替代关系。

小结：
